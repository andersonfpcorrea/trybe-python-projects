# Project Job Insights

## Instructions

1 - Clone this repository

2 - Create virtual environment for the project

```bash
python3 -m venv .venv && source .venv/bin/activate
```

3 - Install dependencies

```bash
python3 -m pip install -r dev-requirements.txt
```

4 - Run linter

```bash
python3 -m flake8
```

5 - Run tests

```bash
python3 -m pytest
```

## Run the project

```bash
flask run
```
